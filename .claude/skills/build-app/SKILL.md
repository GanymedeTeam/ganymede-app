---
name: build-app
description: Run to build the application for production, creating optimized and minified files ready for deployment.
version: 1.0.0
---

To build the application for production, you can use the following command in your terminal:

```bash
pnpm tauri build
```

If the user wants to build a debug version of the application, they can use the following command:

```bash
pnpm tauri build --debug
```
